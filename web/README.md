## Se clonou o repositório do GitHub

Depois de `git clone` e `cd` no repositório, entre em **`web/`** e rode `npm install` e `npm run dev` (detalhes no README da pasta pai).

---

## Comandos rápidos (execute dentro desta pasta `web`)

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).
